CUPS 添加打印机教程

1、打开打印机电源并使用数据线连接到盒子USB接口上

2、进入盒子的首页，并右上角选择局域网，点击打印机图标

   
3、之后浏览器会打开新窗口进入 CPUS 管理页面，选择 Administrator 标签然后点击 Add Printer 按钮进入添加打印机步骤

   

4、点击按钮后稍等几秒，页面会刷新并列出检测到的所有可添加打印机，下面以HP2130打印机为例
   由于惠普打印机有自己的接口协议，所以识别出来两个打印机，其实是同一个，用的协议不一样，如果你也是惠普打印机推荐选择带有 USB 字符的那一个，选择好之后点击 Continue 按钮

   

5、接下来的页面刷新后需要填写相关信息，使用默认信息即可，另外需要在 Share This Printer 处的选择框打勾，然后点击 Continue 按钮

   

6、然后需要选择相应的驱动程序，使用推荐的第一个即可，选择完驱动继续点击 Continue 按钮

   

7、设置打印机的相关参数，一般情况默认即可，然后点击 Set Default Options 按钮完成添加操作

   

8、此时打印机添加完成，建议使用测试功能打印一张测试纸来测试打印功能是否正常
